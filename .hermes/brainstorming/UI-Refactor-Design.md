# 个人网站 UI 重构设计方案 (UI-Refactor-Design)

**目标**: 将现有的“暗黑赛博朋克”风格重构为“明亮、温暖、高级且极具数据/技术感”的极简科技风，同时不改变现有的内容结构与组件摆放顺序。

## 1. 核心视觉哲学 (Visual Philosophy)
- **基调**: 温暖高级的极简科技风，犹如一份正在运行的精密产品蓝图。
- **色彩系统**: 
  - 背景色: 暖调米白 / 浅灰 (例如 `#F5F4F0`)
  - 主文字: 深炭黑 / 墨灰 (例如 `#1A1A1A`)，保证极高对比度
  - 强调色: 亮砖红 / 朱红 (例如 `#E53935`) 与 纯黑，用于点睛之笔。

## 2. 核心元素重塑 (UI Elements)
- **排版 (Typography)**: 
  - 大标题 (Display): 引入**点阵字体 (Dot-matrix/Pixel font)** 展现硬核科技感。
  - 正文 (Body): 保持极其干净现代的无衬线字体，形成强烈的视觉张力。
- **背景与容器**: 
  - 引入极细的**虚线网格 (Dotted grid)** 和交叉标作为暗纹，暗示数据思维。
  - 移除霓虹发光与毛玻璃效果，改为纯色圆角卡片，配合 1px 精确边框或极简硬阴影 (Brutalist shadow)。
- **按钮与标签**:
  - 黑底白字的高对比度设计，部分关键标签采用带有工程感切角或等宽字体 (Monospace) 的参数化设计。

## 3. 动效与微交互 (Motion & Micro-interactions)
- **视差与弹性滚动 (Spring Parallax)**: 卡片入场采用带有阻尼感 (Spring) 的阶梯式滑入；背景网格与前景卡片产生滚动视差，增加空间纵深。
- **高亮个人优势**: 在“AI产品力”、“增长闭环”等核心区域，采用警示感的红色圆点、左侧高亮红线进行视线狙击。
- **磁吸交互 (Magnetic Hover)**: 针对 CTA 按钮 (如 Request a Demo) 和关键技能标签，鼠标悬停时产生轻微磁吸偏移与细微的红色线框扫描动画。
- **视觉连线**: 在经历时间轴与卡片间，采用极细的红色虚线与实心黑点进行连接，展现“逻辑与闭环”的个人特质。

## 4. 实施原则
- 绝对不改变当前的 DOM 结构大纲和内容顺序。
- 所有的改动仅限于 CSS 样式 (如 Tailwind 类名替换)、Framer Motion 参数调整以及少量的背景修饰 SVG/Div 注入。
